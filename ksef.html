InfirSoft/Ksef Last updated: x

Wrapper PHP dla API Krajowego Systemu e-Faktur (KSeF) udostępnianego przez Ministerstwo Finansów.

Api

  • Ksef::sessionAuthorisationChallenge() - Inicjalizacja mechanizmu uwierzytelnienia i autoryzacji
  • Ksef::sessionInitToken() - Inicjalizacja sesji, wskazanie kontekstu, uwierzytelnienie i autoryzacja (za pomocą tokena)
  • Ksef::sessionTerminate() - Wymuszenie zamknięcia aktywnej sesji
  • Ksef::sessionStatusCurrent() - Sprawdzenie statusu aktywnej sesji interaktywnej
  • Ksef::sessionStatus() - Sprawdzenie statusu sesji ogólnej
  • Ksef::queryCredentialSync() - Zapytanie o poświadczenia
  • Ksef::queryInvoiceSync() - Zapytanie o faktury
  • Ksef::invoiceGet() - Pobranie faktury
  • Ksef::invoiceSend() - Wysyłka faktury
  • Ksef::invoiceStatus() - Sprawdzenie statusu wysłanej faktury
